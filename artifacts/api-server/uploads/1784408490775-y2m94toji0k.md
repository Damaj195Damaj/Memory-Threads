# Kestrel Hollow Expedition — Project Notes

**Project code:** AHT-2025-114
**Sponsor:** Ardenmoor Heritage Trust
**Institution:** University of Caldbridge, Dept. of Archaeology
**Dates:** June 2–9, 2025
**Site:** Kestrel Hollow, Ardenmoor Highlands (ruined chapel & monastic outbuildings, est. 15th c.)

## Objective

Locate and recover material evidence relating to the **Thornecliff Manuscript**, a 15th-century illuminated text referenced in the Caldbridge monastery ledger (entry dated 1487) but never recovered. Secondary objective: document the chapel foundation layout ahead of planned Trust conservation work in 2026.
